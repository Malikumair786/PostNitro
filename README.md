- [Phosphor Icons](https://phosphoricons.com/): v2.1.7

## Getting Started

1. Click on the "Use as template" button and create a new repository
2. Clone the project
3. Run commands:
```sh
npm install
npm run dev
```
The project should start at http://localhost:3000
